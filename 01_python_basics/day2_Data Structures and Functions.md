# Day2:  Data Structures and Functions 数据结构与函数

## 1. List
用来存放多个元素的数据结构.
列表使用方括号 `[]` 来表示
```python
scores = [99,85,77,90,66,59]
print(scores)
```

### 访问列表元素
```python
scores = [99,85,77,90,66,59]
print(scores[0])
print(scores[1])
print(scores[-1])
```
### 列表长度
```python
scores = [99,85,77,90,66,59]
print(len(scores))
```

### 添加新元素
```python
scores = [99,85,77,90,66,59]
scores.append(100)
print(scores)
```
